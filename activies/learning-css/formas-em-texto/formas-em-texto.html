Shape-outside

Essa propriedade permite que você defina uma forma para que o texto se envolva e flue ao seu redor.

Junto a esta propriedade temos algumas formas básicas

O elemento que vamos aplicar a forma com a propriedade Shape-outside precisa ter a propriedade Float

  • inset()
  • circle()
  • ellipse()
  • polygon()
    •

    inset()

    Pode ser usado para criar um retângulo/quadrado com um deslocamento opcional para o texto que o envolver.
